# 악보생성기 기획 설계 문서

**날짜:** 2026-04-17  
**대상:** 디자인팀 인계 전 기획 완성본  
**범위:** MVP — 단일 페이지 악보 생성 서비스

---

## 1. 핵심 결정 사항

| 항목 | 결정 |
|------|------|
| 악보 생성 방식 | 하이브리드 (DB 원곡 + AI 수준별 편곡) |
| 곡 검색 방식 | 텍스트 검색 + 자동완성 드롭다운 |
| 사용자 계정 | MVP 불필요 — 로그인 없이 바로 사용 |
| 렌더링 형식 | 악기별 최적 형식 자동 선택 |
| 공유 기능 | MVP 불필요 — PDF 저장만 제공 |
| 페이지 구조 | 심플 원페이지 (페이지 이동 없음) |

---

## 2. 전체 구조 및 사용자 흐름

```
[메인 페이지 — 단일 화면]

① 검색바 (자동완성)
   └─ 곡명 입력 → 후보 드롭다운 표시

② 옵션 선택
   └─ 악기: 드럼 / 기타 / 우쿠렐레 / 피아노 (탭 또는 버튼)
   └─ 수준: 입문 / 초급 / 중급 (토글)

③ "악보 생성" 버튼
   └─ 클릭 → 로딩 (최대 10초)

④ 악보 렌더링 영역 (인라인, 스크롤)
   └─ 악기별 최적 형식 (TAB/오선지/드럼보)
   └─ 재생 중 현재 마디 하이라이트

⑤ 컨트롤 바 (악보 하단 고정)
   └─ 재생 / 일시정지 / 정지
   └─ BPM 슬라이더 (40~200)
   └─ 메트로놈 On/Off
   └─ PDF 저장 버튼 (1일 3회 무료)
```

**흐름 요약:** 검색 → 옵션 선택 → 생성 → 악보+컨트롤 인라인 노출. 페이지 이동 없음.

---

## 3. 데이터 흐름

```
[클라이언트]
  곡명 + 악기 + 수준 입력
        ↓
[검색 API]
  곡명 자동완성 → MIDI/악보 DB 조회
        ↓
[편곡 엔진 (AI)]
  원곡 MIDI 데이터 → 수준별 편곡 처리
  - 입문: 단순화 (코드 수 축소, 리듬 단순화)
  - 초급: 부분 편곡
  - 중급: 원곡 근접 편곡
        ↓
[렌더링]
  MusicXML / TAB 데이터 → 브라우저 악보 렌더링
  MIDI → Web Audio API로 재생
        ↓
[저장]
  악보 → PDF 변환 → 다운로드
  일일 횟수: 브라우저 로컬스토리지로 카운트 (로그인 없음)
```

---

## 4. 예외 처리 정책

| 상황 | 처리 방식 |
|------|----------|
| DB에 곡 없음 | "유사 곡 추천" 리스트 3개 노출 |
| AI 편곡 1~2회 실패 | 재시도 버튼 노출 |
| AI 편곡 3회 이상 실패 | "이 곡은 현재 지원이 어렵습니다. 다른 곡을 검색해보세요" + 검색바 포커스 이동 |
| 생성 10초 초과 | 타임아웃 안내 + 재시도 버튼 |
| PDF 저장 3회 초과 | 구독 유도 모달 ("무제한 저장은 구독으로") |
| 지원하지 않는 악기·수준 조합 | 해당 옵션 비활성화 처리 |

---

## 5. 악기별 렌더링 스펙

| 악기 | 렌더링 형식 | 비고 |
|------|-----------|------|
| 피아노 | 오선지 (treble + bass clef) | 양손 표기, 입문은 오른손만 |
| 기타 | TAB + 오선지 병렬 표시 | 코드 다이어그램 상단 노출 |
| 우쿠렐레 | TAB + 코드 다이어그램 | 오선지 생략 (입문자 친화) |
| 드럼 | 드럼 기보법 (퍼커션 클레프) | 범례(어떤 선이 어떤 드럼인지) 악보 상단 표시 |

**공통 렌더링 규칙:**
- 마디 번호 표시
- BPM · 박자표 · 조표 상단 표시
- 재생 시 현재 마디 하이라이트 (노란색)
- 모바일: 가로 스크롤로 악보 탐색

## 6. MVP 범위 최종 확정

### ✅ 포함
- 곡명 검색 + 자동완성
- 악기 4종 · 수준 3단계 선택
- 하이브리드 악보 생성 (DB + AI 편곡)
- 악기별 최적 렌더링
- 재생 / 일시정지 / 정지
- BPM 슬라이더 (40~200) + 메트로놈 On/Off
- PDF 저장 (1일 3회 무료)
- 예외 처리 (실패 안내, 구독 유도 모달)

### ❌ MVP 이후
- 로그인 / 계정 시스템
- 링크·SNS 공유
- 구간 반복 재생
- 악보 편집
- 악기 추가 확장
- 모바일 앱
