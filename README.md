# 악보생성기

원하는 곡과 악기, 수준을 입력하면 해당 악보를 생성해줌. (드럼, 기타, 우쿠렐레, 피아노)

## 프로젝트 정보
- **유형**: SaaS 도구
- **타겟 사용자**: 입문자, 취미 연주자
- **출시 목표**: MVP 즉시 출시

## 핵심 기능
- 곡명 + 악기 + 수준 입력 시 악보 자동 생성
- 악보 재생 (생성된 악보로 음악 미리듣기)
- 메트로눔 설정

## 폴더 구조 (5개 팀)
- `planning/` — 기획 팀: 무엇을 만들 것인가
- `design/` — 디자인 팀: 어떻게 보여줄 것인가
- `cx/` — CX 팀: 누가 어떻게 사용할 것인가
- `marketing/` — 마케팅 팀: 어떻게 알릴 것인가
- `strategy/` — 전략 팀: 어떻게 지속 가능하게 만들 것인가
