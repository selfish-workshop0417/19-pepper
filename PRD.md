# 악보생성기 — PRD

## 1. 서비스 개요
악보생성기는 입문자와 취미 연주자를 위한 악보 자동 생성 SaaS 도구입니다. 원하는 곡명, 악기(드럼/기타/우쿠렐레/피아노), 연주 수준을 입력하면 맞춤 악보를 즉시 생성해줍니다. MVP는 무료로 제공하며 추후 구독 모델로 전환합니다.

## 2. 폴더 구조
이 레포는 5개 팀(파트)에 따른 5개 영역으로 나뉜다:
- planning/ — 기획 (무엇을 만들 것인가)
- design/ — 디자인 (어떻게 보여줄 것인가)
- cx/ — CX (누가 어떻게 사용할 것인가)
- marketing/ — 마케팅 (어떻게 알릴 것인가)
- strategy/ — 전략 (어떻게 지속 가능하게 만들 것인가)

각 영역의 상세는 해당 폴더에서 작성된다.

## 3. 기획 (Planning)
- 곡명 + 악기 + 수준 입력 → 악보 자동 생성
- 생성된 악보 재생 (미리듣기)
- 메트로눔 BPM 설정
- MVP 범위: 위 3개 기능, 4개 악기 지원
